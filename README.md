1. git status
2. git add [files] - Добовляет файлы в (stage - временное хранилище)
3. git reset [files] - Удаляет файлы из stage
3. git reset --hard - Отменяет все изменения и очищает stage
4. git commit -m "comment" - Создает коммит
5. git log / git log --oneline
6. git push [rep_link origin] [branch_name] - Добовляет на github 
7. git diff - Показать изменение строчек.
